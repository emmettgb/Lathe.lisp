# Lathe.lisp
![Lathe Logo](http://lathe.ai/logo.png)\
Lathe.lisp is a port of the machine-learning module for Julia, Python, and R (Lathe) into common lisp.\
Quick Load:\
**Bash**
```bash
cd ~/
git clone https://github.com/emmettgb/Lathe.lisp.git
```
**C Lisp**
```lisp
(load "path/to/lathe")
```
